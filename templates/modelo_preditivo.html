Metodologia e Resultados

Objetivo

Desenvolver um modelo de machine learning capaz de predizer a probabilidade de hospitalização de pacientes com dengue com base em características demográficas, sintomas clínicos e condições climáticas.

Metodologia

  • Algoritmo: Regressão Logística
  • Dataset de Treinamento: Dataset balanceado com 58.963 registros
  • Features Utilizadas: 5 features (Sertãozinho, Idade, Sexo, Mialgia, Cefaleia, Vômito, Exantema)
  • Pré-processamento: One-hot encoding, normalização, balanceamento de classes
  • Validação: Divisão treino/teste (80/20) com estratificação e random_state=42

Observação sobre o Balanceamento: O dataset de treinamento foi balanceado para resolver o desbalanceamento extremo dos dados reais (1.17% de hospitalização). O balanceamento foi realizado com casos reais de hospitalização do ano de 2025, escolhidos aleatoriamente, sendo metade homem e metade mulher, garantindo que o modelo aprenda a identificar corretamente a classe minoritária. O modelo foi otimizado para maximizar o Recall, o que é crucial em um contexto de saúde pública.
